# VendorBridge

## Procurement & Vendor Management ERP

VendorBridge is a Procurement & Vendor Management ERP system designed to streamline procurement operations through a centralized platform. The system enables organizations to manage vendors, create RFQs, collect quotations, process approvals, generate purchase orders, create invoices, and track procurement activities efficiently.

## Features

### Authentication & Authorization

* User Login & Signup
* Role-Based Access Control
* Forgot Password
* Secure Session Management

### Vendor Management

* Vendor Registration
* Vendor Categories
* Vendor Status Tracking
* GST & Contact Details Management
* Search & Filtering

### Procurement Workflow

* Create RFQs (Request for Quotations)
* Assign Vendors to RFQs
* Receive Vendor Quotations
* Compare Quotations
* Approval Workflow
* Purchase Order Generation
* Invoice Generation

### Reporting & Analytics

* Procurement Statistics
* Vendor Performance Analytics
* Spending Summaries
* Monthly Procurement Trends
* Activity Logs & Notifications

## User Roles

### Admin

* Manage Users
* Manage Vendors
* View Analytics

### Procurement Officer

* Create RFQs
* Compare Quotations
* Generate Purchase Orders
* Generate Invoices

### Manager / Approver

* Approve or Reject Procurement Requests
* Monitor Procurement Workflows

### Vendor

* Submit Quotations
* Track RFQ Status
* View Purchase Orders

## Workflow

1. Procurement Officer creates an RFQ.
2. Vendors receive RFQs and submit quotations.
3. Quotations are compared.
4. Approval workflow is initiated.
5. Approved quotations generate Purchase Orders.
6. Invoices are generated.
7. Invoices can be downloaded, printed, or emailed.
8. Procurement activities are tracked through logs and analytics.
